**プロジェクト名：** 単身赴任の味方アプリ「ストック番長」  
**作成日：** 2026年4月14日  
**最終更新：** 2026年4月14日（v3：買い物日サジェスト機能 追加）  
**想定実装環境：** Claude Code

## 1. プロジェクト概要

### 背景・目的
単身赴任中、卵・プロセスチーズ・PETアイスコーヒーの在庫切れに気づかず毎日スーパーへ足を運んでしまう。その結果、余計なお菓子まで購入するという悪循環を断ち切るための在庫管理アプリを開発する。

### 解決したい課題
- 在庫切れに気づかず買い物頻度が増える
- 買い物のたびに余計なものまで買ってしまう
- 小型冷蔵庫なのでまとめ買いできる量に限界がある

### ゴール
> 「スーパーへ行くのは週2回（月・木）以内」を実現する

## 2. 管理対象アイテムと基本設定値（デフォルト）

| アイテム | 在庫入力単位 | 1日消費量 | 購入単位（1回） | 在庫計算の内部単位 |
|---|---|---|---|---|
| 卵 | 個 | 4個/日 | 1ケース = 10個 | 個 |
| プロセスチーズ | 個 | 3個/日 | 1スリーブ = 4個 | 個 |
| PETアイスコーヒー | 本 | 300ml/日 | 1本 = 900ml（= 3日分） | ml（内部換算） |

> ※ アイスコーヒーは「本」で入力するが、内部では1本=900mlとして日数計算する
> ※ デフォルト値はすべて設定画面から変更可能

---

## 3. コアロジック：在庫日数の計算

### 定義
```
在庫日数 = 現在の在庫量 ÷ 1日消費量
```

### 各アイテムの計算式

| アイテム | 計算式 | 例（在庫5個/本の場合） |
|---|---|---|
| 卵 | 在庫（個）÷ 4個/日 | 5 ÷ 4 = **1.25日** |
| チーズ | 在庫（個）÷ 3個/日 | 5 ÷ 3 = **1.67日** |
| アイスコーヒー | 在庫（本）× 900ml ÷ 300ml/日 | 2本 × 900 ÷ 300 = **6日** |

---

## 4. コアロジック：アラート判定

### 買い物サイクル（デフォルト：月・木）
- システムは「今日から見て次の買い物予定日」を自動計算する
- 例）今日が火曜日 → 次の買い物日は「木曜日（2日後）」

### ステータス判定ロジック

```
在庫日数 < 次の買い物日までの残り日数
  → 🔴 ALERT（今すぐ買わないと次の買い物日前に切れる）

在庫日数 < 次の買い物日までの残り日数 × 1.5倍（余裕バッファ）
  → 🟡 WARNING（そろそろ準備が必要）

上記以外
  → 🟢 OK（余裕あり）
```

### 具体例（今日が火曜日、次の買い物日が木曜＝2日後）

| アイテム | 在庫 | 在庫日数 | ステータス | 判定理由 |
|---|---|---|---|---|
| 卵 | 3個 | 0.75日 | 🔴 | 0.75 < 2日 → 今日切れる |
| チーズ | 8個 | 2.67日 | 🟡 | 2.67 < 3.0（2×1.5）→ 余裕なし |
| アイスコーヒー | 3本 | 9日 | 🟢 | 9 ≥ 3.0 → 余裕あり |

### 特別アラート：「今日が買い物日なのに在庫切れ間近」
- 今日が買い物日（月・木）かつ🔴アイテムがある場合
- → ホーム画面最上部に赤バナーで強調表示
- → LINE通知にも「今日が買い物日です！」を追記

## 5. 機能要件

### 5-1. 在庫入力機能（最重要）

**入力方式：** 数字入力（スマホテンキーが開く）

**入力タイミング：**
- 毎日20時のLINE通知を受け取った後
- 買い物から帰宅した直後

**入力内容：**
- 各アイテムの「現在の在庫数」を直接入力（絶対値入力）
- 消費量のトラッキングは不要（負担軽減のため在庫数のみ）
- 「今日の購入数」は任意入力（記録用）

**UI要件：**
- スマートフォン最適化（タップしやすいボタンサイズ 44px以上）
- −/＋ボタン or 数字直接入力
- 入力欄の下に「この在庫で あと X日分」をリアルタイム表示
- 「保存」1タップで完了、トースト通知でフィードバック

### 5-2. ホーム画面（在庫一覧＋アラート）

**表示内容：**
- 各アイテムのステータスカード（3枚）
  - アイテム名・現在の在庫数・在庫日数・ステータス色
- 「次の買い物日：〇曜日（X日後）」表示
- 🔴アイテムがある場合：「今日買うもの」バナーを上部に赤表示

### 5-6. 買い物日サジェスト機能（新設）

**概要：**  
買い物曜日（月・木）に在庫を更新した後、「次の買い物日まで欠品しない購入量」を自動計算して提案する機能。スーパーに着く前に「何を何個買えばいいか」が明確になる。

**発動条件：**
- 今日が設定した買い物曜日（デフォルト：月・木）であること
- 在庫更新画面で「保存」をタップした直後に自動表示

**計算ロジック：**

```
必要量     = 1日消費量 × 次の買い物日までの日数
不足量     = 必要量 − 現在の在庫量
購入推奨数  = ceil(不足量 ÷ 購入単位) × 購入単位  ← 単位に合わせて切り上げ
```

**具体例（月曜日に入力・次の買い物日は木曜＝3日後）：**

| アイテム | 現在庫 | 3日間の必要量 | 不足量 | 購入推奨 |
|---|---|---|---|---|
| 卵 | 3個 | 4×3＝12個 | 9個 | **1ケース（10個）** |
| チーズ | 2個 | 3×3＝9個 | 7個 | **2スリーブ（8個）** |
| コーヒー | 1本(900ml) | 300×3＝900ml | 0ml | **購入不要 ✅** |

> ※ アイスコーヒーは「本」で入力→内部でml換算して計算し、結果は「本」で表示する

**不足がないアイテムの表示：**
- 計算結果が0以下の場合は「購入不要 ✅」と表示
- 全アイテム購入不要の場合は「今日は買い物不要です！ ✅」バナーを表示

**UI仕様（ポップアップ形式）：**

```
╔══════════════════════════════════╗
║  🛒 今日の買い物リスト           ║
║  （木曜まで 3日分）              ║
║                                  ║
║  🥚 卵　　 1ケース（10個）       ║
║  🧀 チーズ 2スリーブ（8個）      ║
║  ☕ コーヒー 購入不要 ✅         ║
║                                  ║
║  [ LINEに送る ]  [ 閉じる ]      ║
╚══════════════════════════════════╝
```

**「LINEに送る」ボタンの動作：**
- サジェスト内容をテキストに変換してLINEへ送信
- スーパーの中でLINEトーク画面を開いて確認できる形式

**LINE送信テンプレート：**
```
【ストック番長】今日の買い物メモ 🛒
木曜まで（3日分）

🥚 卵　　 1ケース（10個）
🧀 チーズ 2スリーブ（8個）
☕ コーヒー 購入不要 ✅

忘れずに！
```

## 9. 在庫日数 → アラート判定アルゴリズム（実装メモ）

```javascript
// 次の買い物日までの残り日数を計算
function getDaysUntilNextShoppingDay(shoppingDays) {
  const today = new Date().getDay(); // 0=日〜6=土
  const sorted = [...shoppingDays].sort((a, b) => a - b);
  for (const day of sorted) {
    const diff = (day - today + 7) % 7;
    if (diff > 0) return diff;
  }
  return 7 - today + sorted[0]; // 翌週の最初の買い物日
}

// ステータス判定
function getStockStatus(stockDays, daysUntilShopping) {
  if (stockDays < daysUntilShopping)        return "ALERT";   // 🔴
  if (stockDays < daysUntilShopping * 1.5)  return "WARNING"; // 🟡
  return "OK";                                                 // 🟢
}

// アイスコーヒーの在庫日数計算（本 → ml換算）
function getCoffeeDays(bottleCount, settings) {
  const totalMl = bottleCount * settings.purchaseUnit.coffee.amount; // 本 × 900ml
  return totalMl / settings.dailyConsumption.coffee.amount;          // ÷ 300ml/日
}

// 汎用：在庫日数計算
function getStockDays(item, count, settings) {
  if (item === "coffee") return getCoffeeDays(count, settings);
  return count / settings.dailyConsumption[item].amount;
}

// 買い物日サジェスト：アイテムごとの購入推奨量を計算
function getSuggestedPurchase(item, currentCount, daysUntilNextShopping, settings) {
  const daily = settings.dailyConsumption[item].amount;
  const unitSize = settings.purchaseUnit[item].amount;

  // アイスコーヒーはml換算して計算
  const currentAmount = (item === "coffee")
    ? currentCount * settings.purchaseUnit.coffee.amount  // 本 → ml
    : currentCount;

  const needed = daily * daysUntilNextShopping;           // 必要量
  const shortage = needed - currentAmount;                // 不足量

  if (shortage <= 0) return { count: 0, label: "購入不要" };

  const units = Math.ceil(shortage / unitSize);           // 購入単位で切り上げ
  const totalAmount = units * unitSize;

  // アイスコーヒーは本に戻して返す
  const displayCount = (item === "coffee")
    ? Math.ceil(totalAmount / settings.purchaseUnit.coffee.amount)
    : totalAmount;

  return { count: displayCount, units: units };
}
```
